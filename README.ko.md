## 📖 개요

Grow Kitty는 게임화를 통해 일상 루틴을 매력적인 경험으로 변화시키는 혁신적인 오픈소스 습관 관리 애플리케이션입니다. 더 나은 습관을 만들어가며 가상 고양이가 성장하고 번창하는 모습을 지켜보세요!

매일 제공되는 랜덤 미션, 의미 있는 보상, 그리고 당신의 성취에 반응하는 사랑스러운 가상 고양이 동반자를 결합하여, Grow Kitty는 습관 형성을 재미있고 상호작용적이며 감정적으로 보람 있게 만듭니다.

## 🎯 왜 Grow Kitty인가요?

기존 습관 추적 앱들은 반복적인 디자인과 감정적 연결 부족으로 인해 장기적인 사용자 참여를 유지하는 데 실패하는 경우가 많습니다. Grow Kitty는 다음과 같은 방법으로 이러한 문제를 해결합니다:

- **감정적 참여**: 가상 고양이가 당신의 진행 상황에 반응하여 의미 있는 유대감 형성
- **역동적인 게임플레이**: 매일 제공되는 랜덤 미션으로 신선하고 흥미로운 경험 유지
- **소셜 연결**: 친구들과 성과를 공유하고 경쟁
- **시각적 진행**: 개인 발전과 함께 성장하는 고양이 관찰

## ✨ 핵심 기능

| 기능 | 설명 |
| --- | --- |
| 🐈 **가상 고양이 성장** | 습관 달성률에 따라 고양이가 진화하여 시각적 동기 부여 제공 |
| 🎯 **랜덤 일일 미션** | 매일 새로운 도전 과제로 참여도 유지 및 단조로움 방지 |
| 🎁 **보상 시스템** | 앱 내 보상 획득, 귀여운 액세서리 잠금 해제, 고양이 커스터마이징 |
| 👥 **커뮤니티 기능** | 진행 상황 비교, 성과 공유, 리더보드 순위 상승 |
| 📊 **습관 대시보드** | 성공률, 연속 달성 일수, 상세한 진행 통계 추적 |
